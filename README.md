# 🧹 Insurance Data Cleaning Project

مشروع تنظيف بيانات التأمين الصحي باستخدام Python و Pandas

---

## 📋 وصف المشروع

هذا المشروع يقوم بتنظيف بيانات التأمين الصحي من الأخطاء والقيم الناقصة والمكررة، وتجهيزها للتحليل أو بناء نماذج الذكاء الاصطناعي.

---

## 📁 محتويات المشروع

```
insurance-data-cleaning/
├── clean_insurance.py       # سكريبت التنظيف الرئيسي
├── insurance.csv            # البيانات الأصلية
```

## 📊 البيانات

| العمود | النوع | الوصف |
|--------|-------|-------|
| `age` | int | عمر المؤمَّن عليه |
| `sex` | str | الجنس (male / female) |
| `bmi` | float | مؤشر كتلة الجسم |
| `children` | int | عدد الأطفال المعالين |
| `smoker` | str | هل هو مدخن؟ (yes / no) |
| `region` | str | المنطقة الجغرافية |
| `expenses` | float | تكاليف التأمين الصحي |

---

## 🔧 خطوات التنظيف

1. **حذف الصفوف المكررة** — تم حذف 1 صف مكرر
2. **تنظيف أسماء الأعمدة** — إزالة المسافات والرموز الغريبة
3. **عمود age** — التحقق من النطاق (18-100) وملء الناقص بالوسيط
4. **عمود sex** — توحيد القيم إلى male / female
5. **عمود bmi** — التحقق من المنطقية وإضافة عمود التصنيف
6. **عمود children** — التحويل إلى integer وحذف القيم السالبة
7. **عمود smoker** — توحيد القيم إلى yes / no
8. **عمود region** — تنظيف النصوص وتوحيد الشكل
9. **عمود expenses** — حذف القيم السالبة وإضافة عمود التصنيف
